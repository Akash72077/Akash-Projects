SDE: System design engineerning

Scaling:
Scaling means increasing the capacity of a system so it can handle more users, more requests, or more data.

Imagine you own a restaurnt.

initially you have 10 customers/day

later you have 10,000 customers/data

then your old setup won't be enough.

Now you have two choices:

1)make your restaurnt bigger.(vertical Scaling)
2) Open more restaurnts.(Horizontal Scaling)

These are exactly the two types of Scaling

1)vertical Scaling(Scale Up)

Defination:

            vertical scaling means increasing the power of one server.
            Instead of adding more servers, you improve the existing one.

            Example:

                Before:

                    server
                        CPU : 4 Cores
                        RAM : 8 GB
                        Storage : 250 GB
            
                Upgrade it to: 

                    server

                        CPU : 32 Cores
                        RAM : 128 GB
                        Storage : 4 TB SSD

            here server is same but power is improvee
